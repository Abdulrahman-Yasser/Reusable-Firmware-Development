Detailed Description

Description:

This function is used to initialize the SPI based on the configuration table defined in SPI_cfg.c module.

PRE-CONDITION: Port_Init() should be called before any action to initialize the pins on the SoC PRE-CONDITION: PLL_Init() should be called before any action to initialize the MCU clock PRE-CONDITION: Configuration table needs to populated (sizeof > 0)
PRE-CONDITION: NUMBER_OF_DEFINED_SPI > 0
PRE-CONDITION: NUMBER_OF_BOARD_SPI_CHANNELS > 0
PRE-CONDITION: The MCU clocks must be configured and enabled. POST-CONDITION: The SPI peripheral is set up with the configuration settings.

Parameters
[in]SPIConfigsis a pointer to the configuration table that contains the initialization for the peripheral.
Returns
void

Example:

const SPIConfig_t *SPIConfig_t = SPI_ConfigGet();
Spi_Init(SpiConfig_t);
